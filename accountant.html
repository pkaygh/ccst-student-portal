CSV Format Requirements

Required Columns:

  • id - Student Index Number (e.g., CCST/RM/24/001)
  • INDEX NUMBER - Alternative student ID column
  • NAME - Student's full name
  • Course columns: Course codes as headers (e.g., CCC 016, CCN 012)

Optional Columns:

  • PROGRAM - Program code (e.g., RM, ML)
  • LEVEL - Academic level (e.g., 100, 200)
  • SEMESTER - Semester identifier

Course Structure:

For each course, include columns for CA (Continuous Assessment) and EXAM scores.
Example: CCC 016, CA, EXAMS
